Häufig gestellte Fragen zum Herunterladen von Minecraft

WAS IST DER MINECRAFT LAUNCHER?Mit dem Minecraft-Launcher können Sie auf alle Ihre Lieblingsspiele auf Ihrem Computer zugreifen. Mit dem Minecraft-Launcher können Sie neue Versionen von Minecraft-Spielen herunterladen, auf Multiplayer-Server zugreifen, Ihr Profil verwalten, Modifikationsunterstützung erhalten und vieles mehr. Weitere Informationen zum Launcher finden Sie im Support-Center.

WIE INSTALLIERE ODER INSTALLIERE ICH DEN MINECRAFT LAUNCHER?Sie können den Minecraft-Launcher über den Microsoft Store installieren oder oben auf dieser Seite auf die Schaltfläche „Launcher herunterladen“ klicken. Wenn Sie den Launcher neu installieren müssen, gehen Sie einfach zum Microsoft Store und laden Sie ihn erneut herunter. Wenn Sie eine ältere Version des Minecraft-Launchers haben, verwenden Sie unsere Anleitung zum Deinstallieren von Minecraft Unified Launcher, um die neueste Version zu erhalten.

WIE AKTUALISIEREN SIE DEN MINECRAFT LAUNCHER?ann Sie Ihren Minecraft-Launcher aktualisieren müssen, erfahren Sie in Ihren Kontoeinstellungen. Gehen Sie zu Einstellungen > Alle verfügbaren Launcher-Updates finden.

IST ES MÖGLICH, MINECRAFT OHNE DEN MINECRAFT LAUNCHER ZU STARTEN?Wenn Sie Minecraft ohne Minecraft-Launcher spielen möchten, dann ist dies noch kein Spiel, alle wichtigen Spiele an einem Ort.
